# サンプルデータ → 統合CSV 転記可否まとめ

## 前提：サンプルデータの種類

クライアントから受け取ったサンプルデータは以下の3種類。

| ファイル種別 | 何が書いてあるか |
|------------|----------------|
| 品質仕様書 PDF（15件） | 品名・成分・物流情報 |
| 価格交渉記録 PDF（20件） | 値上げ理由・交渉経緯 |
| 単価改定履歴 XLS（13件） | 改定日・改定前後の価格 |

---

## ○ 転記できるもの（サンプルデータから取れる）

| カラム | 転記元 | 補足 |
|--------|--------|------|
| 文書番号 | 品質仕様書 PDF | |
| 品名 | 品質仕様書 PDF | |
| 分類番号（大/中/小/商標） | 品質仕様書 PDF | |
| 運送経路 | 品質仕様書 PDF | 例：静岡→土浦 |
| 所要時間 | 品質仕様書 PDF | 例：6時間前後 |
| 運送会社 | 品質仕様書 PDF | |
| 成分名（成分1〜10） | 品質仕様書 PDF | ほぼ全品目で取れる |
| 成分の含有量（%） | 品質仕様書 PDF | ※下の「△」欄を参照 |
| 改定後の価格 | 単価改定履歴 XLS | |
| 改定時期 | 単価改定履歴 XLS | |
| 改定前の価格 | 単価改定履歴 XLS | 前回の単価から算出 |
| 改定理由 | 価格交渉記録 PDF | 改定時期が一致するものだけ紐付け可能 |

---

## △ 品目によって取れる・取れないもの

### 成分の含有量（%）

品質仕様書に「成分名だけ」書いてある品目と「%まで」書いてある品目が混在している。

| 状態 | 品目 |
|------|------|
| **%まで書いてある**（転記可） | AC-238、AC-288、EP P-12N、プルーフセメントNR、ハイドリック VS EX、カネライトフォーム、LBT-733、GW33126、TR2034N1、K-75Z、ZEST PQ140、オクサゾールS |
| **成分名だけで%なし**（別資料が必要） | **WK-680RIP(965)**（SDS等が必要） |
| **成分名自体がない** | **セラフォーム TR150**（仕様書に化学組成の記載なし） |

> PoC では WK-680RIP の%はクライアント提供のシミュレーターPDFの値（78%/15%/10%）をダミーとして使っている。

---

## × 転記できないもの（外部データが必要）

これらは**サンプルデータのどこにも記載がない**。外部から別途取得する必要がある。

| カラム | 入手できない理由 | 本来の入手先 |
|--------|----------------|-------------|
| 輸送距離（km） | 仕様書に経路・時間はあるがkm数の記載なし | Google Maps等で手動算出 |
| 材料1〜10・含有量 | サンプル仕様書にはこの区分の記載なし | 別途ヒアリングまたは別資料 |
| **成分の市況単価** | 外部市況データ。社内書類に含まれない | gdfreak.com、経産省統計等 |
| **燃動力の種別・単価** | 交渉記録に「天然ガス高騰」等の言及はあるが具体的な単価は記載なし | 外部市況サイト |
| **労務費（時間当たり）** | 書類のどこにも記載なし | 厚労省 賃金構造基本統計等 |
| **運賃（円/m）** | 書類のどこにも記載なし | 運送会社見積もり、トラック協会統計等 |

> 太字の4項目がシミュレーションの核心。これらが外部データなしでは「ダミー値でのデモ」にしかならない理由。

## 価格改定履歴がない品目

以下の2品目は改定履歴XLSが存在しないため、価格情報が一切ない。

- **リューロンペースト K-75Z**（東ソー）
- **ZEST PQ140**（新第一塩ビ）

## まとめ：どこまでが「自動」でどこからが「手作業」か

```
【自動で取れる（LLMで抽出可能）】
  品名・文書番号・成分名・物流情報・価格改定履歴・値上げ理由
  ↓ サンプルデータ52件から抽出済み

【品目によって取れない（別資料が必要）】
  成分の含有量%（WK-680RIPなど一部品目）
  ↓ SDS（安全データシート）等を入手できれば抽出可能

【外部データがないと絶対に埋まらない】
  市況単価・燃動力費・労務費・運賃
  ↓ PoCではダミー値で代替
  ↓ 本番では外部APIまたは手動入力の仕組みが必要
```
